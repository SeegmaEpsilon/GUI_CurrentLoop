Инструкция актуальна для версии прошивки датчика 3.2.2.
Для настройка датчика необходим мультиметр, ОБЯЗАТЕЛЬНО ЗАЗЕМЛЕННЫЙ блок питания, USB-UART преобразователь (желательно на чипе CH340 или FTDI, НЕ ИСПОЛЬЗОВАТЬ PL2303), ПК и ПО ВД17 Сервис.

============================ ИНИЦИАЛИЗАЦИЯ ============================
1 Закрепите датчик на вибростоле.

2 Подведите провода питания к датчику, подключите по схеме, само питание не включайте. 

3 Подсоедините переходник USB-UART сначала к компьютеру, потом к датчику.

4 В программе, в выпадающем окне рядом с надписью "COM-Порт", выберете COM-порт используемого UART-преобразователя.

5 Нажмите кнопку "Подключиться".

6 Дочитайте инструкцию и подайте питание на датчик. 
ВНИМАНИЕ: Блок питания должен быть ОБЯЗАТЕЛЬНО заземлен и настроен на 24.0 вольта.

7 В случае успешного подключения в консоли приложения будет отрисовываться график и появятся данные.
Если этого не произошло, начните с шага 2.

============================ НАСТРОЙКА ТИПА ИЗМЕРЕНИЙ ============================
8 Выберете параметр, которое будет измеряться датчиком.
Выберете из выпадающего списка поля "Измеряемый параметр для ТП" необходимый параметр и нажмите кнопку "Применить".

9 Выберете ось, которая будет измеряться датчиком.
Выберете из выпадающего списка поля "Измеряемая ось для ТП" необходимый параметр и нажмите кнопку "Применить".

10 Выберете значение виброскорости, которое будет соответствовать значению тока в 20.00 мА.
Введите в поле "Предельное значение" необходимое число и нажмите кнопку "Применить".

11 Выберете динамический диапазон для измерений датчика.
Выберете из выпадающего из выпадающего списка поля "Динамический диапазон" необходимый параметр и нажмите кнопку "Применить".

============================ НАСТРОЙКА ПРЕДЕЛОВ ТОКОВОЙ ПЕТЛИ ============================
12 Откалибруйте НИЖНИЙ предел токовой петли.
Для этого необходимо подобрать такое число, при котором значение тока на миллиамперметре будет равно 4.00 миллиампера (мА):
12.1 В поле "Постоянная виброскорость" введите значение 0.001 и нажмите кнопку "Применить". После этого, на ШИМ токовой петли будет идти значение, заданное из поля "Нижний предел ТП (код)".
12.2 Сравните текущее значение тока на миллиамперметре с необходимым (4.00 мА), если ток больше необходимого значения, уменьшите число в поле "Нижний предел ТП (код)", если меньше - увеличьте число и снова нажмите "Применить".
12.3 Повторяйте процедуру, пока ток на мультиметре не будет равен 4.00 +- 0.01 мА.

13 Откалибруйте ВЕРХНИЙ предел токовой петли.
Для этого необходимо подобрать такое число, при котором значение тока на миллиамперметре будет равно 20.00 миллиампер (мА):
13.1 В поле "Постоянная виброскорость" введите значение 200 и нажмите кнопку "Применить". После этого, на ШИМ токовой петли будет идти значение, заданное из поля "Верхний предел ТП (код)".
13.2 Сравните текущее значение тока на миллиамперметре с необходимым (20.00 мА), если ток больше необходимого значения, уменьшите число в поле "Верхний предел ТП (код)", если меньше - увеличьте число и снова нажмите "Применить".
13.3 Повторяйте процедуру, пока ток на мультиметре не будет равен 20.00 +- 0.01 мА.

14 Верните датчик в боевой режим.
После настройки верхнего и нижнего предела ОБЯЗАТЕЛЬНО запишите 0 в поле "Постоянная виброскорость" и нажмите кнопку "Применить", иначе датчик будет выдавать на токовую петлю одно и то же значение и не будет ничего измерять.

============================ КАЛИБРОВКА ОСЕЙ ДАТЧИКА ============================
15 Откалибруйте оси датчика.
15.1 Расположите датчик так, чтобы гравировка оси "X" смотрела вверх.
15.2 Запустите формирование сигнала на вибростоле (10.10 м/с2, 159.15 Гц).
15.3 Дождитесь выравнивания графиков в сервисе и выхода на уставку.
15.4 Выберете из выпадающего списка поля "Калибруемая ось" параметр "X" и нажмите кнопку "Применить".

15.1 Расположите датчик так, чтобы гравировка оси "Y" смотрела вверх.
15.2 Запустите формирование сигнала на вибростоле (10.10 м/с2, 159.15 Гц).
15.3 Дождитесь выравнивания графиков в сервисе и выхода на уставку.
15.4 Выберете из выпадающего списка поля "Калибруемая ось" параметр "Y" и нажмите кнопку "Применить".

15.1 Расположите датчик так, чтобы гравировка оси "Z" смотрела вверх.
15.2 Запустите формирование сигнала на вибростоле (10.10 м/с2, 159.15 Гц).
15.3 Дождитесь выравнивания графиков в сервисе и выхода на уставку.
15.4 Выберете из выпадающего списка поля "Калибруемая ось" параметр "Z" и нажмите кнопку "Применить".

============================ ПРОВЕРКА РАБОТЫ ДАТЧИКА ============================
16 Проверьте работу датчика в боевом режиме.
16.1 Расположите датчик так, чтобы гравировка оси "Z" смотрела вверх.
16.2 Запомните/запишите значение из поля "Предельное значение"
16.3 Введите в поле "Предельное значение" число 20 и нажмите кнопку "Применить".
16.4 Запустите формирование сигнала на вибростоле (10.00 м/с2, 159.15 Гц).
16.5 Ток на мультиметре должен составлять 12.00 +- 0.01 мА.
16.6 Введите в поле "Предельное значение" число из пункта 16.2 и нажмите кнопку "Применить".
